VK Photo Uploader

Веб-интерфейс для загрузки фотографий в ВКонтакте

О программе

Это веб-приложение позволяет загружать фотографии в альбом ВКонтакте с использованием CSV файла для структурирования данных.

Конфигурация

Загрузите config.txt с настройками доступа

CSV файл

photos.csv с описанием фото и комментариев

Фотографии

Все изображения для загрузки

Как использовать:

  1. Подготовьте файлы (config.txt, photos.csv, фотографии)
  2. Загрузите их через форму
  3. Дождитесь обработки
  4. Получите результат
