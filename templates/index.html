### 3. Фотографии Все фотографии должны быть в одной папке с config.txt и CSV файлом. ## Как получить VK Access Token 1. Создайте Standalone приложение VK: https://vk.com/editapp?act=create 2. Получите ключ доступа с правами: - `photos` - `groups` - `wall` - `offline` 3. Получите `GROUP_ID` и `ALBUM_ID` из URL страницы альбома ## Развертывание на Render 1. Создайте новый Web Service на Render 2. Подключите этот репозиторий 3. Установите переменные окружения: - `SECRET_KEY` - случайная строка для сессий 4. Запустите приложение ## Использование 1. Откройте веб-интерфейс приложения 2. Выберите папку со всеми файлами (config.txt, CSV, фотографии) 3. Нажмите "Проверить файлы" 4. Нажмите "Начать загрузку в VK" 5. Следите за прогрессом в логе ## Технические детали - Файлы не сохраняются на сервере Render - Потоковая обработка файлов до 4 МБ каждый - Автоматическая разбивка комментариев на группы по 2 фото - Отказоустойчивость при ошибках сети ## Ограничения - VK API имеет лимиты на количество запросов - Максимум 2 фото в комментарии (ограничение VK) - Бесплатный план Render: 512 МБ RAM, 100 МБ диска
