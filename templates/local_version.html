Локальная версия

Для обработки больших объемов фотографий на вашем компьютере

На главную

Почему локальная версия?

  • Без ограничений по количеству фотографий
  • Без ограничений по времени выполнения
  • Работает с тысячами фотографий
  • Не зависит от интернета после загрузки
  • Полный контроль над процессом

Скачать локальную версию

Архив содержит все необходимое для работы на вашем компьютере

Скачать vk-photo-uploader-local.zip

Инструкция по установке:

  1. Скачайте архив и распакуйте его
  2. Откройте терминал/командную строку в папке с программой
  3. Установите зависимости: 
    pip install -r requirements.txt
  4. Создайте файл config.txt с токеном ВКонтакте
  5. Создайте файл photos.csv с данными фотографий
  6. Поместите фотографии в ту же папку
  7. Запустите программу: 
    python main.py

Дополнительные параметры:

Для больших объемов используйте параметры командной строки:

# Указать папку с фотографиями
python main.py --folder "C:\Мои фото"

# Настроить размер пакета (рекомендуется 10-20)
python main.py --batch 15

# Увеличить задержку между пакетами (рекомендуется 30-60 для больших объемов)
python main.py --delay 45

# Все параметры вместе
python main.py --folder "C:\Фото" --batch 20 --delay 60
